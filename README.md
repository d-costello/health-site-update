# This repository
An example and proof of concept website update for the DC Health Respiratory Disease pages

## Goal
The original goal was to update the language on the site to a more accessible and plain language for the general public to quickly understand.
Additional information on healthcare provider requirements was also needed.

It became quickly apparent that the current site is distinctly outdated. This mock design and implementation is a proof of concept of what could be done in the future

## Process
index.html is the respiratory home page that links to the disease information pages (RSV, COVID 19, Influenza, and legionellosis). Additional resources for healthcare workers is under caregiver.html (empty at the moment for future input based on discussions)

styles.css houses the design and graphical elements of the page
script.js is needed to change pages 

## Required extensions
- live server to display the code
